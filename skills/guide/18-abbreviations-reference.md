## Abbreviations Reference

Maple uses abbreviations for CSS properties. If a property is not listed, use its camelCase version (e.g., `scrollBehavior`).

### Layout & Display

| Abbreviation | Property   |
| ------------ | ---------- |
| `d`          | display    |
| `pos`        | position   |
| `t`          | top        |
| `r`          | right      |
| `b`          | bottom     |
| `l`          | left       |
| `z`          | zIndex     |
| `of`         | overflow   |
| `ofx`        | overflowX  |
| `ofy`        | overflowY  |
| `v`          | visibility |
